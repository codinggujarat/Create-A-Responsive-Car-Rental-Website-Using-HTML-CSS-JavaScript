## 🛠️ Tech Stack

- **HTML5** – Semantic markup  
- **CSS3** – Modern styling with Flexbox & Grid  
- **JavaScript (ES6)** – Interactivity and animations  
- *(Optional: GSAP & Locomotive Scroll for smooth animations)*
